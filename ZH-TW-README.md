KK_PerspectiveX

此款適用於《戀愛》的第一人稱視角插件，不會讓攝影機穿著角色肢體，也不易造成暈厥。

可用於主遊戲的自由移動模式、H場景，以及CharaStudio。

同時支援 Koikatsu 與 Koikatsu Sunshine，兩隻遊戲共用一個 DLL 檔。我已經在 Sunshine 中進行過完整測試，目前的運作表現與原版 Koikatsu 幾乎相同。不過，由於 Sunshine 是較新且見識的第一人稱支援目標，偶爾仍可能遇到 Sunshine 獨有的錯誤。

如果未來遊戲的差異變得更大，可能需要將插件分割成不同的版本，但目前同一個 DLL 能夠同時支援兩種遊戲。


為什麼還需要另一個第一人稱視角插件？

目前已有的第一個人稱視角插件主要有兩個問題。

部分插件將攝影機放置的位置錯誤，導致視角位於胸口高度，而且角色頭部會穿進螢幕。

RealPOV雖然能夠正確放置攝影機，但它會完整複製頭部的動畫旋轉，其中也包含左右傾斜，因此畫面會隨著每個動畫動作左右歪斜、晃動。

PerspectiveX 將攝影機的位置與旋轉分開處理。

以上內容是根據閱讀 RealPOV 與 KK_StudioPOV 的公開原始碼所整理，僅在說明程式碼的實際運作方式中，並非在貶低任何一個專案。

安裝方式 需要安裝 BepInEx 5，HF Patch 已經內建。 從 Releases 下載 KK_PerspectiveX.dll，把檔案放在： BepInEx/plugins/ 如果你已經安裝 RealPOV，我認為 HF Patch 可能已經內建了，附帶： RealPOV.Koikatu.dll

重新命名為：

RealPOV.Koikatu.dll.已停用

這樣可以終止RealPOV，因為這兩個插件都使用Backspace作為第一人稱視角切換鍵。

已在 Koikatsu 與 Koikatsu Sunshine 搭配 BepInEx 5 以上版本進行測試與站立。

如果在您的遊戲環境中無法正常使用，請提出問題回饋，我會查看問題。

操作方式

所有按鍵都可以透過ConfigurationManager，預設按F1開啟，重新進行設定。

退格：開啟或關閉第一人稱視角。在工作室中，必須先在工作區選擇一個角色。按住鼠標左鍵並拖曳：轉動視角。未拖曳時，鼠標遊標仍會顯示並可正常操作。保持 Ctrl + L：可實現免按住鼠標的 FPS 視角控制。鼠標遊標會被鎖定，直到再次按 Ctrl + L。 Ctrl + Shift + 左方向鍵/右方向鍵：將第一人稱視角切換至上一個或下一個角色。鼠標滾輪：在第一人稱視角中經常調整FOV。若不小心滾輪，可在設定中關閉此功能。逗號鍵／號鍵：讓攝影機向左或向右傾斜。斜線鍵可將傾斜角度重設為水平。 分號鍵：將攝影機鎖定在目前位置。鎖定後，視角不會再靠近頭部移動，特別適合讓頭部鎖定大幅晃動的愛撫動作。其間仍可自由轉動視角。再次按下後即可鎖定解除，攝影機會平滑移回角色頭部位置。 Ctrl + Shift + 1/2/3：將目前視角儲存至指定欄位，FOV、方向觀看、傾斜角度與攝影機偏移。

Ctrl + 1/2/3：載入對應欄位中的已儲存視角。

儲存位置欄會保留，即使關閉遊戲後再次啟動也不會消失。

Ctrl + P：開啟預設面板。在 Studio 中，也可以點選左下角工具列中的 PerspectiveX 眼睛按鈕。預設面板

遊戲內面板將所有與預設相關的功能集中在同一個位置，不需要再進入插件設定中查找。

視角預設設定：提供一鍵套用的設定組合，包括：

舒適 60 自然 90 動作 110

這些預設值會同時調整FOV、位置平滑與攝影機偏移。

自訂預設：可自行調整喜歡的FOV、平滑程度與偏移組合，輸入名稱後儲存成自己的單鍵預設，最多可儲存5組。 已儲存視角：與鍵盤相同的3個儲存與載入欄位，會以按​​鈕顯示，並顯示每個欄位儲存的FOV。

舒適性設置，包括位置平滑、動畫晃動、FOV 與攝影機偏移，都可以在插件設置中調整，並且可以在第一人稱模式中即時應用。

視角預設也可以直接在插件設定中使用。

存在一個可選設定：

「攝影機姿勢」

啟用後，攝影機會性行為身體的方向定向。例如角色側躺時，畫面相似身體方向定向，而不是強制保持地平線水平。

從原始碼構建

此插件以.NET Framework 3.5為目標框架，可以在任何作業系統上使用一般的.NET SDK進行編譯。

首先，請從遊戲安裝目錄複製以下參考DLL檔案至專案的lib/資料夾。

由於這些文件受版權保護，因此不會包含在原始碼專案中。

來自以下資料夾：

BepInEx/核心/

複製：

BepInEx.dll 0Harmony.dll

來自以下資料夾：

Koikatu_Data/Managed/

複製：

Assembly-CSharp.dll Assembly-CSharp-firstpass.dll UnityEngine.dll UnityEngine.UI.dll

隨後執行：

cd src/KK_PerspectiveX dotnet build -c Release

編譯完成後，輸出檔位於：

src/KK_PerspectiveX/bin/Release/KK_PerspectiveX.dll
